How to Use
How to set boundaries:
  • Step 1: Click a drawing tool (Circle, Rectangle, or Polygon)
  • Step 2: Draw the boundary directly on the map
  • Step 3: Fill in the location details and name
  • Step 4: Click "Save Boundary" to apply
  • Step 5: Use "Test Boundary" to verify the area
  • Step 6: Toggle between different map views as needed

Tip: For circles, you can adjust the radius in the input field after drawing

Note: The default radius is 40m, but you can set any value between 5m and 5000m
